<h1 align=center>双目测距</h1>

最新代码请查看[dev](https://openi.pcl.ac.cn/3DVision/StereoMeasurement/src/branch/dev)分支。

本项目使用普通的双目RGB相机测量目标距离。

[参考视频](https://www.youtube.com/watch?v=xjx4mbZXaNc)


## 依赖

### 实物：

+ 普通RGB双目相机

+ 棋盘格标定板

### 运行环境:

+ python 3.7.5

+ numpy
+ openCV
+ openpyxl
+ scikitlearn
